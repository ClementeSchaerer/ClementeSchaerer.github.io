Ejercicio Fusion 360 - Mano

Utilizando el programa Fusion 360, llevamos a cabo un tutorial utilizando un modelo de mano ya existente. En base a este modelo creamos un yeso para la muñeca. Para esto utilizamos los siguientes comandos: “offset plane”, “sketch”, “loft”, “thicken” y, por último, el comando “fillet”.

Utilizando los comandos ya mencionados, pudimos crear el yeso para el brazo a partir de nuestro dibujo a mano alzada, creando los planos necesarios para luego dibujar el contorno del modelo. Luego recubrimos el modelo uniendo los dibujos y modificando las caras y puntos necesarios para que, finalmente, agregar el grosor y curvatura correspondiente al yeso.
