Ejercicio Fusion 360 - Sketch

Utilizando el programa Fusion 360, llevamos a cabo un tutorial para crear un invernadero a partir de un dibujo a mano alzada.

Utilizamos los comandos: “sketch”, “pipe”, “mirror” y, por último, el comando “pattern”.

Utilizando los comandos mencionados anteriormente, se realizó el dibujo a mano alzada, sobre el cual se construyeron los pipes necesarios para los tubos del modelo, los cuales fueron luego replicados.

Finalmente, se replicó el modelo en su totalidad para luego ponerle un recubrimiento, el cual se le brindó la característica de ser transparente por ser un invernadero.
