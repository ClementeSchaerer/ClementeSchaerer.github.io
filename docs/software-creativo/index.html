Software Creativo: Runway

Tutorial Runway: Herramienta “Green Screen”

En este tutorial se muestra el programa RunwayML, versión beta, el cual utiliza inteligencia artificial para su funcionamiento. Esto quiere decir que el programa comienza a aprender y absorber la información de todos los proyectos creados en este para luego poder perfeccionar sus herramientas para los futuros usuarios y proyectos.

En este video tutorial se presenta un paso a paso de como utilizar una de las herramientas más significativas de Runway, Green Screen. Esta herramienta permite incorporar una pantalla verde a cualquier video de manera amigable y fácil de utilizar.
